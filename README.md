# Time4Break

## Setup:
### Dependencies:

```
python -m pip install opencv-python numpy mediapipe
```

### Build and Run:
```
git pull https://github.com/AndreSlavescu/Time4Break
```
```
python drowsy.py
```

### Features:

- Nose direction tracking
- FPS counter
